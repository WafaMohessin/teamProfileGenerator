# teamProfileGenerator
This is a Node.js command-line application which takes the employees' information on a software engineering team, and generates an HTML webpage that displays a brief for each person.

# Installation
- npm i
- node index

# Techonologies Used
- HTML
- CSS
- Javascript
- Node JS
- Jest
